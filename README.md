# GroomAI – AI-Powered Salon Booking

AI-powered grooming marketplace for discovering and booking salon services.

## Tech Stack
- **Frontend:** Next.js 16.2.9 (App Router), React 19, Tailwind CSS v4, TypeScript
- **Auth:** Clerk (`@clerk/nextjs`)
- **Backend:** FastAPI (Python) deployed on Vercel
- **Database:** Supabase (PostgreSQL)
- **AI:** OpenAI-compatible API via Ollama Cloud
- **Payments:** Razorpay

## Deployed URLs
- **Frontend:** https://frontend-one-amber-82.vercel.app
- **Backend API:** https://backend-ten-ashen-42.vercel.app
- **API Docs:** https://backend-ten-ashen-42.vercel.app/docs

## Setup
1. Clone the repo.
2. Copy `.env.example` to `.env.local` and fill in credentials.
3. Run `npm install` and `npm run dev`.
4. Open `http://localhost:3000`.

## Testing
Run `npm test`.
